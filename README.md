# Networking Labs Portfolio

## Purpose
This repository is a showcase of networks I have designed from scratch using Cisco Packet Tracer.

These labs demonstrate foundational and intermediate networking concepts I have learned over the years, and have been created to showcase my understanding of these concepts.

## Contents
Each folder in this repository contains a separate networking lab. Each lab includes:
- Packet Tracer topology file (.pkt)
- Network design documentation (.pdf)
- README to quickly explain the purpose and functionality of the lab/topology.
